🚀 COMECE AQUI (3 passos simples):

   1️⃣  Configure sua API:
       └─ Abra: lib/services/product_service.dart
       └─ Altere a linha: static const String baseUrl = 'https://sua-api.com'

   2️⃣  Instale dependências:
       └─ flutter pub get

   3️⃣  Execute:
       └─ flutter run
